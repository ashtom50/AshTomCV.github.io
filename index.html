Eink ToDo List

Design
  • Raspberry Pi Zero was soldered to the Eink Display.
  • Python code was written to extract ToDoIst API and display it to the Eink display.
  • A case was 3D printed to house the final product.

My current ongoing Pi project is using an eink display to show my to do list from the Todoist app. I purchased a 4.2- inch Waveshare eink display and use a Pi zero w to control it. I started with the sample python code provided by Waveshare to test the screen. As I understood more of the code, I cut more and more of it down until I had only the bare essentials that I could use as a starting point. From the Todoist documentation I used their rest API to download my current to do list and filter them down to just todays and overdue tasks. These are then displayed on the screen alongside their due dates. Once all the tasks are completed the screen then shows my tasks for the next seven days. I am now making a 3D printed case, adding a power button and wiring the Pi to the back of the screen directly for a clean finished product.

I have completed many other Raspberry Pi projects over the last few years. Some of these include a network attached storage device, a server for remote control of a 3D printer (discussed below), as a home network advertisement blocker, retro gaming console and as a security camera using Motioneye.
